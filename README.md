## 🎯 Ce que vous allez apprendre

- ✅ Pourquoi l'interface graphique **(UI/GUI) ne protège pas** votre application
- ✅ Comment un attaquant peut **décompiler votre APK** avec JADX ou apktool
- ✅ Comment configurer les **Firebase Security Rules** pour bloquer tout accès non autorisé
- ✅ Comment empêcher l'**interception des requêtes** (attaque Man-in-the-Middle)
- ✅ Comment protéger votre app contre les **scripts Python directs** sur l'API Firebase

---

## 🛠️ Technologies abordées

| Technologie | Usage |
|---|---|
| Flutter / Dart | Développement de l'application mobile |
| Firebase Auth | Authentification sécurisée des utilisateurs |
| Firebase Firestore | Base de données temps réel sécurisée |
| Firebase Security Rules | Barrière serveur incontournable |
| ProGuard / R8 | Obfuscation du code APK |

## 👨‍💻 Pour qui est ce guide ?

Ce guide est fait pour vous si :
- Vous débutez avec Flutter et Firebase
- Vous avez une app en production mais n'avez **pas encore configuré les Firebase Rules**
- Vous voulez comprendre **concrètement** comment fonctionne une attaque sur une app mobile
- Vous cherchez des **règles prêtes à copier-coller** pour sécuriser votre Firestore
